== Description ==

= The best free plugin to display the Australia Post live rates =

Use this free Australia Post WooCommerce Live Rates plugin to offer your customers the Australia Post shipping methods for domestic and international shipping.

**Don't waste your time**, enter your Australia Post credentials and **set everything up in your shop effortlessy in just 5 minutes**!

Show your clients the Australia Post shipping services with their prices being dynamically calculated in the cart and checkout. Once the Australia Post API connection is established, the **shipping cost for each Australia Post service is calculated real-time** based on the products' weight, shop location and delivery destination.

= Features =

* Automatic shipping cost calculation and displaying the Australia Post live rates in the cart and checkout
* Domestic and international Australia Post services
* Australia Post services' shipping cost calculation based on the cart weight and the shipping destination
* Limiting services only for those available for the customer's address
* Manual Australia Post services limiting and possibility to choose which services should be displayed and which not
* Possibility to set the fixed fallback amount in case no Australia Post rates were returned
* Free shipping over amount threshold
* Package default dimensions and weight
* Optional excluding the GST from Australia Post shipping rates
* Dedicated debug mode for easy troubleshooting

== Installation	==

This plugin can be easily installed like any other WordPress integration by following the steps below:

1. Download and unzip the latest zip file release.
2. Upload the entire plugin directory to your **/wp-content/plugins/** path.
3. Activate the plugin using the **Plugins** menu in WordPress sidebar menu.

Optionally you can also try to upload the plugin zip file using **Plugins** &rarr; **Add New** &rarr; **Upload Plugin** option from the WordPress sidebar menu. Then go directly to point 3.

== Frequently Asked Questions ==

= How can I configure Australia Post services? =
To determine which Australia Post services should be available for the customers to choose, use the **Services: Enable the services custom settings** option and select from the Services Table the ones you want to be displayed.
